# athevon-skills

Collection de skills Claude Code — git workflow, code quality, creative coding, design.

## Installation

```bash
/plugin marketplace add git@github.com:AThevon/claude-skills.git
/plugin install athevon-skills@AThevon
```

## Skills inclus

### Git workflow
- **pr** — Crée une PR complète (rebase, push, create)
- **push** — Add, commit et push les changements
- **rebase** — Rebase la branche actuelle sur main
- **update-pr** — Met à jour le titre et la description d'une PR existante
- **release-github** — Release GitHub complète et automatique
- **bump** — Bump la version du projet

### Code quality
- **build** — Lance le build et corrige les erreurs
- **fix-ci** — Analyse et corrige les erreurs de CI
- **lint** — Vérifie et corrige les erreurs de lint importantes
- **lint-all** — Corrige toutes les erreurs de lint
- **review** — Review complète du code
- **review-short** — Review concise du code
- **clean-comments** — Nettoie les commentaires inutiles

### Productivity
- **copy-env** — Copie les .env depuis le worktree principal
- **readme** — Met à jour le README selon les derniers changements

### Creative & Design
- **creative** — Mode expert en creative coding
- **creative-3D** — Mode expert en creative coding 3D/animations avancées
- **ui-ux-pro-max** — UI/UX design intelligence (50 styles, 21 palettes, 9 stacks)
- **design-excellence** — Pipeline design d'exception
